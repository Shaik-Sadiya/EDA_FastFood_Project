# Exploratory Data Analysis on Fast Food Nutrition Dataset

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a fast-food menu dataset to identify nutritional trends, calorie distribution, protein content, and category-level insights.

## Objectives

- Data Cleaning
- Descriptive Statistics
- Nutritional Analysis
- Visualization
- Recommendations

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
