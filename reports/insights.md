# Fast Food Nutrition Dataset Insights

## Key Findings

### 1. Calories Analysis
- Some menu items contain very high calories.
- Calorie distribution is not uniform.

### 2. Category Analysis
- Certain categories have much higher average calories.
- Some categories are comparatively healthier.

### 3. Protein Analysis
- High-protein foods generally have higher calories.

### 4. Sodium Analysis
- Several menu items contain excessive sodium levels.

### 5. Correlation Analysis
- Calories show positive correlation with fat.
- Protein and calories are moderately related.

## Recommendations

1. Promote low-calorie menu items.
2. Reduce sodium in highly processed products.
3. Highlight high-protein healthy foods.
4. Display nutrition information clearly.
5. Create healthier meal combinations.
